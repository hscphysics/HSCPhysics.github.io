==================

1. User opens index.html
2. Browser loads CSS (styles.css)
3. Browser loads JavaScript (main.js, flashcards.js, etc.)
4. main.js loads home.html into content area
5. User clicks navigation â†’ loads different page
6. JavaScript initializes features for that page
7. Everything works offline, no server needed!
